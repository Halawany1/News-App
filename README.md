# News App (Flutter)

this app shows the news of Business , Sports ,Science.
you can choose dark or light mode .
you can search about News do you want and you can go to the website and read the news .

# Tools
- Flutter & Dart
- Api
- Dio package
- Shared preferences
- Bloc (cubit)
- Web view
